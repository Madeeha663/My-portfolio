# My Portfolio Website

## Overview

This is my personal portfolio website developed to showcase my skills, projects, and background as a BS Information Technology student at Bahria University Islamabad.

The website highlights my technical skills, academic projects, and contact information in a clean and responsive design.

* Responsive design
* Professional user interface
* About Me section
* Skills section
* Projects showcase
* Contact information
